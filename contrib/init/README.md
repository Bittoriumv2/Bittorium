Sample configuration files for:
```
SystemD: bittoriumd.service
Upstart: bittoriumd.conf
OpenRC:  bittoriumd.openrc
         bittoriumd.openrcconf
CentOS:  bittoriumd.init
macOS:    org.bittorium.bittoriumd.plist
```
